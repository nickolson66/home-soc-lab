# home-soc-lab
Built a home Security Operations Center lab using Splunk, Sysmon, Kali Linux, and Windows VMs.

## Tools Used
- Splunk
- Sysmon
- Kali Linux
- Windows 10 VM
- VirtualBox

## What I Configured
- Log forwarding with Splunk Universal Forwarder
- Endpoint telemetry with Sysmon
- Detection queries using SPL
- Dashboards for process and network activity

## Example Detections
- Port scanning
- Suspicious PowerShell activity
- Failed login attempts

## Skills Demonstrated
- SIEM Monitoring
- Threat Detection
- Log Analysis
- Incident Investigation
